# Description

Exercice to create a table of user lists and add actions:

- Delete row ✔️
- Reset table ✔️
- Sort options (Name, Last name, Country and Time)

  - Name ✔️
  - Last Name
  - Country
  - Time

- Group options
- Pagination
- Get users from API

- Add chart map pointing with dots where are the users
